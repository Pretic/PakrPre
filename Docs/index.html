当前部署说明以 README 为准

PakrPre 已在原 Pakr 基础上加入管理密码、自定义域名链接、默认图标、访问身份、元素屏蔽、规则导入导出和详细日志等功能。旧版文档中的部分部署步骤已经不再适用。

重要差异
  • GitHub PAT 应配置到 Cloudflare Pages 环境变量：GH_PAT 或 GITHUB_TOKEN。
  • GitHub Actions Secrets 只放签名相关字段：KEYSTORE_BASE64、KEYSTORE_PASSWORD、KEY_ALIAS、KEY_PASSWORD。
  • ADMIN_PASSWORD 不会自动生成，只有手动添加后才启用访问保护。
  • PUBLIC_BASE_URL 可选；设置后从 pages.dev 访问会跳转到自定义域名。

常用入口

部署教程查看仓库 README 中的最新完整流程。 构建日志进入 GitHub Actions 查看完整构建记录。 Cloudflare Pages配置 Pages 变量、域名和重新部署。 GitHub Token创建或更新 GitHub Personal Access Token。
